# Automation - CRUD de Produtos (Foco em QA)

Este projeto Full-Stack (React, Node.js, PostgreSQL) serve como um laboratório prático para a aplicação de conceitos de Quality Assurance (QA) e desenvolvimento de testes automatizados (SDET).

## 🛠️ Como Executar o Projeto

Abra **3 terminais independentes** e execute os comandos na ordem abaixo:

### 1. Banco de Dados

```bash
sudo systemctl start postgresql
```

### 2. Backend

```bash
cd backend
npm install
npm run dev
```

### 3. Frontend

```bash
cd frontend
npm install
npm run dev
```

Acesse a aplicação no navegador através do link gerado no terminal (geralmente `http://localhost:5173`).

---

## 🧪 Testes Automatizados

O projeto utiliza **Vitest** e **React Testing Library** para validação de componentes e fluxos de formulário no frontend.

Para rodar os testes - Frontend:

```bash
cd frontend
npm test
```

Para rodar os testes - Backend:

```bash
cd backend
npm test
```
